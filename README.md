# News Web Scraper

## Project Description

This project scrapes:
- Headlines
- URLs
- Dates
- Summaries

from NPR News website using:

1. requests + BeautifulSoup
2. Playwright + BeautifulSoup

The scraped data is stored in:
- CSV files
- SQLite databases

---

## Technologies Used

- Python
- requests
- BeautifulSoup
- pandas
- sqlite3
- Playwright

---

## Install Requirements

pip install requests
pip install beautifulsoup4
pip install pandas
pip install playwright

playwright install

---

## Run Static Scraper

python static_scraper.py

---

## Run Dynamic Scraper

python dynamic_scraper.py

---
